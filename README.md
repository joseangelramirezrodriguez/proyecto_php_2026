# Proyecto PHP 
## Descripcion 
En este repositorio encontraremos los archivos index.php y una imagen-png en el index esta construida una pagina donde se pueden visualizar la imagen y un **hello world** 
 ## Prerequisitos
 1. Tener instalado una IDE (vs code de preferencia)
 2. Tener sistema Linux o una virtualpc
 3. Tener los siguentes paquetes de instalacion
     - Apache2
     - php
     - git

## Pasos para su ejecucion
1. Abrimos nuestra IDE
2. Abrimos una virtualpc(de no tener un sistema linux nativo en el dispositivo)
3. Abrimos en la terminal de nuestra IDE 
4. Instalamos los paquetes corespondientes
    - Primero hacemos una actualizacion de los paquetes con el comado **sudo apt update**
    - Instalamos apache2 **sudo apt install apache2**
    - Instalamos git **sudo apt install git-all**
    - Instalamos php **sudo apt php**
5. Iniciamos apache2 con el siguente comando en la terminal **sudo systemctl start apache2**
6. Ubicamos el archivo html con /var/www/html
7. Creamos una carpeta 
8. Vinculamos nuestra cuenta de git
9. Clonamos el repositorio con el siguente comado **git clone (aqui va la url del repositorio)**
10. Para poder visualizar los archivos con apache2 usamos la ip/nombre de la carpeta/nombre del archivo a visualizar(index.php)
